[활용방법]
- useInput : form 작업용 input
- useTabs : 메뉴 등 탭 사용하기 용이하도록 지원.
- useTitle : react document의 title 변경
- usePageLeave : 사용자가 해당 페이지를 떠나려고 할 때를 감지하고 특정 함수 실행.
- useClick : 특정 element 클릭하는 시점을 감지.
- useFadeIn : 모든 element에 적용 가능한 fade in 애니메이션.
- useFullscreen : 모든 element에 적용 가능한 풀스크린 <-> 일반크기 모드
- useHover : 마우스가 위에 올라왔을 때 감지. :hover
- useNetwork : online인지 offline인지 감지.
- useNotification : notification API를 활용하여 사용자에게 알림 기능.
- useScroll : 스크롤 업/다운할 때를 감지. 
- usePreventLeave : 변경사항을 저장하지 않고 떠나려는 등 사용자가 페이지를 벗어나려고 할 때를 감지.
- useConfirm : 특정 조건에서 confirm 받기 
- useAxios : HTTP 요청 클라이언트 axios 기능 활용.
