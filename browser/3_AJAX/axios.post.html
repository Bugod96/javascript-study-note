axios 라이브러리를 활용한 POST 요청

1) axios 라이브러리를 부르는 script:src 코드 입력 후

2-1) axios.post 함수의 첫 번째 인자로 POST 요청을 보낼 url 주소를 입력

2-2) 두 번째 인자에는 요청을 보내면서 서버로 전송할 데이터를 입력


(axios.get과의 공통점)

axios.post의 내부에는 new Promise가 포함. then과 catch 메서드 사용 가능.

then 메서드의 매개변수 result에는 config, data, headers, request, status, statusText 속성들이 포함된 객체가 담기게 됨.

result.data에는 서버로부터 받은 데이터가 담기게 됨.
