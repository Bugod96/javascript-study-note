Query : 데이터베이스에 요청할 구체적인 정보를 GraphQL 언어로 기술한 내용.
- GrqphQL 백엔드에 Query를 전송하고, 그에 맞는 JS 객체 데이터를 응답받음.
- get data (Read)
- 그 자체로는 아무런 기능이 없음. 그냥 설명문.
- 데이터베이스로 보내는 일종의 질문. => Resolver를 통해 해결되어야 하는 질문.

=> 스키마에서는 typeDefs에 Query들 지정
============================================
형식: [query명: return될 데이터의 타입]
1) Int, String만이 아니라 Nicolas처럼 새로운 데이터타입 정의 가능.
ex) 사용자가 num query를 요청하면 Int타입을 응답해준다. (! : isRequired)
2) 쉼표, 주석이 없음. not JS
