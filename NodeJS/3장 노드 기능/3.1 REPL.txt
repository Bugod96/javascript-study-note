[REPL 사용하기]
Javascript는 스크립트 언어이기 때문에 컴파일 없이 즉석에서 코드 실행 가능.

REPL(Read Eval Print Loop) : Node.js의 콘솔창. (브라우저 개발자도구의 콘솔탭과 비슷)
- Read : 입력한 코드를 읽고
- Eval : 해석하고(evaluate)
- Print : 결과물 반환하고
- Loop : R->E->P->R를 종료할 때까지 무한반복
--------------------
콘솔(맥/리눅스/VSC의 터미널, 윈도우의 명령 프롬프트)에서 REPL 사용 방법

REPL 실행 방법
: [node] 엔터 
: 프롬프트가 > 모양으로 바뀌었으면 JS 코드 입력 가능.

REPL 종료 방법
: [.exit] 엔터 // [ctrl]+[C] 두번 실행(^C^C)
